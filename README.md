# Luontopeli

Android-sovellus, jossa käyttäjä voi seurata kävelyä, nähdä reitin kartalla, ottaa luontokuvia, tunnistaa kuvan sisältöä ML Kitillä ja tallentaa löydöt paikallisesti sekä Firebaseen.

## Toteutetut ominaisuudet

Perustoteutuksessa on tehty kaikki viikkojen 1–7 perusominaisuudet:
- Material Design 3 käyttöliittymä ja navigaatio
- askelmittari ja kävelytilastot
- GPS seuranta ja OpenStreetMap
- CameraX kuvanotto
- ML Kit tunnistus
- Discover ja Statsnäkymät
- splash screen ja adaptive icon

## Lisäominaisuudet

Perustoteutuksen lisäksi on tein seuraavat lisäominaisuudet:
- Firebase integraatio (Firebase Auth + Firestore)
- löytöjen näyttäminen kartalla markkereina
